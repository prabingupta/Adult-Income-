# Adult Income Prediction

This project predicts whether an individual's income exceeds $50K per year using machine learning techniques based on the Adult Income dataset.

## Files Included
- `23048504_Prabin_Kumar_Gupta(Income Dataset).ipynb` – Data preprocessing, training, and evaluation
- `train_data.csv` – Training dataset
- `validation_data.csv` – Validation dataset
- `experiment_data.csv` – Experimental dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
